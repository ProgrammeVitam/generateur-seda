VITAM - Générateur SEDA - documentation
####################################################

Cet outil permet de générer des SIP contenant des bordereaux sous forme de fichiers XML, conformes au standard SEDA 2.1 (`lien <http://www.archivesdefrance.culture.gouv.fr/seda/>`_) de manière semi-automatique (au delà d'editeurs XML comme oxygen ou Eclipse) à partir d'une arborescence normalisée de fichiers.
